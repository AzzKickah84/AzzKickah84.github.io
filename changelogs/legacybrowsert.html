LegacyBrowsert

Retro MSHTML browser for Windows 10/11 (separate Windows ME edition available).

Version 1.2.0

June 2026

  • Windows 10/11 WinForms shell using the MSHTML (Trident) engine.
  • Bookmarks, settings persistence, toolbar, and TLS bridge hook for HTTPS via a LAN helper.
  • Installer: LegacyBrowsert-Setup-1.2.0.exe (.NET 8).

Windows ME edition

v1.1.0

  • Standalone HTA / LegacyBrowsert-ME.exe launcher for Windows ME and 9x.
  • Optional LegacyBrowsert-ME-Setup installer built with NSIS 2.46.
