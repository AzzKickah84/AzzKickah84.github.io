Release history and notable changes

Latest: v3.3.18 © Thijs Boekholt — 2026
All notable changes to OutClook are documented here. The format follows Keep a Changelog, and this project uses Semantic Versioning.

18 June 2026
Added

3.3.14

21 June 2026, 02:29
Added
