# Changelog

All notable changes to Galgje (Hangman) are documented in this file.

## [7.3.6] - 2026-06-28

### Fixed
- **Stats set switching** — Notes and game log now properly switch to the correct set's files when changing active stats set
- File paths (hangman_log.txt, notes.txt, played_words.txt) now dynamically update to point to the new active set's directory

## [7.3.5] - 2026-06-28

### Fixed
- **Players menu** now shows all player management options again (was only showing "Show Stats" because menu items were being moved to Tools submenu, which removed them from their original menu)
- **BigBoots auto-loading on startup** — Removed automatic loading of default_players.ini on startup. Players list now starts empty when program launches (users explicitly add players via UI)

## [7.3.4] - 2026-06-28

### Added
- Players menu repopulated with player management options
- New "Show Stats of Selected Player" option in Players menu — displays stats for the currently selected player from either the game list or stats tab
- All player actions (Add/Remove/Eliminate/Mark Winner/Set Errors/Un-eliminate/Reset) back in Players menu

### Fixed
- Status bar (bottom-left) text color changed from light gray to dark blue for much better readability

## [7.3.3] - 2026-06-28

### Added
- All player actions now accessible from Tools menu: Add/Remove/Eliminate/Mark Winner/Set Errors/Un-eliminate/Reset Stats
- Player management (Player Manager, Compare, Export Stats) also added to Tools menu for quicker access
- New "Player Actions" submenu under Tools for cleaner organization

## [7.3.2] - 2026-06-28

### Fixed
- Fixed the missing "TO" (destination) panel — both player lists are now always visible side by side
- The Copy button now actually works (it was a placeholder before), with an option to reset stats to zero on copy
- Move now asks for confirmation and reports exactly which players transferred

### Improved (Stats Sets Manager — full redesign)
- Single, no-tabs screen: pick players on the left (FROM), choose destination on the right (TO), and use the big Copy / Move buttons in the middle
- Authentic Windows 2000 look: classic navy-to-blue gradient title banner, etched separators, sunken 3D list/borders, silver (#D4D0C8) background
- The FROM set doubles as the set you manage — New / Rename / Delete / Activate all act on it, so there is one obvious place for everything
- Live player counts under each list and a clear "★ Active Set" indicator

## [7.3.1] - 2026-06-28

### Fixed
- Fixed NullReferenceException when opening Stats Sets Manager
- Enhanced Windows 2000 styling with proper groupboxes, beveled borders, and classic colors (#C0C0C0)
- Improved tab layout with better spacing and professional groupbox containers
- Status bar now shows active set name in blue at bottom

## [7.3.0] - 2026-06-28

### Improved (Major UX Redesign)
- **Tabbed interface** — Two tabs: "Manage Sets" for set administration, "Copy or Move Players" for player operations
- **Visual copy/move workflow** — Side-by-side player lists with arrow buttons (→ Copy →, → Move →) making the workflow immediately clear
- **Better set management** — Manage, create, rename, delete, and activate sets all in one clean tab
- **Windows 2000 classic aesthetic** — Proper beveled borders, GroupBoxes, classic gray background (#C0C0C0)
- **Live preview** — Right-side player list updates as you select target set, showing exactly what will be there
- **Multi-select support** — Ctrl+Click multiple players for batch operations
- **Much simpler workflow** — No more nested dialogs, everything is in the main window

## [7.2.0] - 2026-06-28

### Improved
- **Stats Sets management** is now much more intuitive with side-by-side player list and Copy/Move buttons
- **Windows 2000/ME aesthetic** with colorful blue header panel (#0066CC) and improved button layout
- **Multi-select players**: Hold Ctrl/Shift to select multiple players to copy or move at once
- **Clear feedback**: Visual labels show exactly which set you're managing and how many players are in it
- **Dialogs redesigned**: All player management dialogs now match the classic Windows 2000/ME style

## [7.1.1] - 2026-06-28

### Improved
- **Stats Editor / Manager**: Added set switcher dropdown — seamlessly switch between sets without closing the dialog
- **Stats Editor grid**: Now shows "Editing {Set} — {N} players" for clarity

- **User experience**: Better feedback and affordances in stats-management workflows

## [7.1.0] - 2026-06-28

### Added
- **Stats Sets** (Tools → Stats Sets): keep independent groups of players, each with their own statistics, and switch between them instantly. The "Default" set holds your existing stats.
- **Stats Editor / Manager** (Tools → Stats Editor / Manager): view and edit every player's games, wins, eliminations, errors and guesses for the active set; add or remove players.
- **Move Player Between Sets** (Tools → Move Player Between Sets): relocate a player to another set without changing any of their statistics.
- **Live statistics in the status bar** (bottom-left): shows the active set, player count, total games, players in the current game and wrong guesses. The copyright/version stays bottom-right.

### Changed
- **All copy/export functions moved from the Edit menu to the Tools menu**, grouped under a single **Copy & Export** submenu. The Edit menu now holds Refresh Statistics and Clear Game Log.
- Application settings (language, shortcuts, preferences) are now shared across all stats sets.
