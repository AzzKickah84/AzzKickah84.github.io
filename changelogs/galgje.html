# Changelog

All notable changes to Galgje (Hangman) are documented in this file.

## [6.8.6] - 2026-06-17

### Added
- **Menu bar icons** using Segoe MDL2 Assets glyphs on all menu items
- **English word list** (`english_wordlist.txt`) for playing with English random words (bundled offline + online fallback)
- **Word list language** setting — choose Dutch (OpenTaal) or English words independently from the interface language
- **View menu** submenus for interface language and word list language (with checkmarks)
- **Copy Achievements Report** in the Edit menu

### Changed
- **About dialog** redesigned with app icon, feature list, and interface/word-list language display
- **Settings dialog** — separate controls for interface language and word list language
- Version bumped to **6.8.6**

## [6.8.5] - 2026-06-17

### Added
- **Classic menu bar** with Game, Edit, Players, View, Tools, and Help menus (keyboard shortcuts included)
- **Settings dialog** (Tools → Settings) with General, Timer, Sound, and Display tabs
- **About dialog** (Help → About Galgje)
- **Persistent game settings** saved to `%LocalAppData%\Galgje\game_settings.ini` (language, sound, timer defaults, hoster name, and more)
- **Menu Bar Edition installer track** — `GalgjeSetup_MenuBar_v6.8.5-menu.1.exe` via `build-release-menubar.ps1` (separate AppId and installer version from the standard setup)

### Changed
- Version bumped to **6.8.5** across application and standard Inno Setup
- Timer bonus/penalty and default duration are now configurable in Settings

## [6.8.4] - 2026-06-16

### Added
- Programming Workshop batch release packaging (installer + press release + changelog in Releases folder)

### Changed
- Version bumped to 6.8.4 across csproj and Inno Setup
