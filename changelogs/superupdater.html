SuperUpdater Changelog

Release history for the Windows system update manager

Current v1.6.5

June 16, 2026

  • Programming Workshop batch release — unified installer packaging in Releases\2026-06-16
  • Press release and changelog bundled with installer distribution
  • Version metadata synchronized across csproj, Inno Setup, and build scripts

Previous v1.6.4

June 16, 2026

  • Manual scheduled run now refreshes the UI after completion (scan no longer skipped while busy)
  • Scheduled Windows Update steps report success/failure correctly in silent runs
  • Scheduled scan-only runs return exit code 0 instead of failing silently
  • Cancelling an operation now terminates the child process tree (winget, choco, pnputil, etc.)
  • Operation cancellation tokens disposed properly between runs
  • Winget agreement flags read from in-memory settings instead of reloading from disk each call
  • Log and status brush converters use frozen cached brushes (lower memory churn)
  • Dashboard welcome title bound to app version (no hardcoded version string in XAML)

June 15, 2026

  • System Restore wizard layout — white content area, navy header rule, overlapping orange/blue accent squares
  • Left task pane with classic radio-button navigation (Dashboard, Programs, Drivers, Problem Hardware, and more)
  • Green accent bar, classic gray footer with 3D buttons, orange corner decoration
  • Splash, About, and Run summary dialogs restyled to match the Windows ME maintenance wizard
  • All existing features preserved — scan, update everything, profiles, CLI, drivers, problem hardware

June 15, 2026

  • Tahoma-style typography, green classic progress bars, wizard title band

v1.4.0
