June 2026

Version 1.2.7

June 2026

  • New installer: Checklist-Setup-1.2.7.exe.

Version 1.2.6

June 2026

  • Live checklist progress counter (enabled items checked vs. total).
  • Pre-save validation warns about missing details, wrong WSEM/LTEM prefix, incomplete items, or not-done status.
  • Unsaved-changes prompt when exiting with modified checklist data.
  • Ctrl+S keyboard shortcut for Save checklist.
  • About dialog layout fixed; 2026 overhaul note added while keeping original 2009–2010 credits.
  • New installer: Checklist-Setup-1.2.6.exe.

Version 1.2.5

June 2026

  • New Tools → Email settings menu with manual SMTP, POP3, and IMAP configuration.
  • Email settings saved to MailSettings.ini with structured sections (legacy format still supported).
  • Fixed checklist save failures caused by invalid filename characters and blocked C:\ root writes.
  • Offline saves now go to Documents\Check Lists.
  • New installer: Checklist-Setup-1.2.5.exe.

Version 1.2.4

June 2026

  • Proper Windows installer (Checklist-Setup-1.2.4.exe) via Inno Setup with uninstall support.
  • Centralized helper module for paths, Excel COM cleanup, versioning, and email reminders.
  • Reliable Excel save/open with error handling and proper COM object release.
  • Window title and About dialog now read version from the assembly automatically.
  • Changelog and press release HTML documents included with the application.
  • Dell service-tag link opens in the default browser instead of hard-coded Internet Explorer.
  • Installer email lookup simplified with case-insensitive name matching.
  • SMTP reminder emails report failures instead of failing silently.
  • Fixed save failure when the user name contains slashes from the date (e.g. NO NAME 17/06/2026).
  • Offline saves go to Documents\Check Lists instead of C:\ (root is blocked on modern Windows).

  • Dell service-tag link for quick support lookup from the serial number field.
  • Desktop (WSEM) and laptop (LTEM) modes with context-specific checklist items.
  • Done / In Progress status buttons with visual indicators.
